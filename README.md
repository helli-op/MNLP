# MNLP
 Каждая компания имеет уникальную корпоративную культуру и процессы. Новым сотрудникам сложно быстро вникнуть во все нюансы, а у руководителей и бадди нет ресурса времени на полноценный онбординг. Существующие wiki-страницы часто разрастаются, теряют структуру и становятся сложными для восприятия. Мы предлагаем  интеллектуального RAG-ассистента (Retrieval-Augmented Generation), который поможет новым и действующим сотрудникам быстро находить ответы на вопросы по корпоративным процессам, документации и регламентам. Ассистент самостоятельно ищет релевантную информацию в базе знаний и формирует понятные ответы.
